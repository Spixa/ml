# بخش ۱: دسترسی به ستون‌ها 

### هدف این بخش
خواندن و تغییر مقدار ستون‌های هر رکورد.

```c
double get(const DataPoint *dp, int col); 
void set(DataPoint *dp, int col, double value);
```


هر سطر دیتا شامل ۹ مقدار عددی است.  
با این توابع می‌توان بدون درگیر شدن با ساختار داخلی، به هر ستون دسترسی داشت.
