		
# بخش ۱: نقش کلی این تابع

### هدف این بخش

نرمال‌سازی داده‌ها قبل از انجام **رگرسیون** با استفاده از **Robust Scaling**.

# بخش ۲: آماده‌سازی ساختار scaler

### هدف این بخش

ذخیره اطلاعات لازم برای scale کردن هر ستون.


### توضیح

برای هر ستون:

- یک **median**
- و یک ‍‍IQR‍
ذخیره می‌شود تا scaling به‌صورت ستونی انجام شود.
# بخش ۳: استخراج داده‌های هر ستون

### هدف این بخش

کار کردن ستونی روی داده‌ها (نه سطری).

### توضیح

برای هر ستون، تمام مقادیر آن ستون در یک آرایه موقت کپی می‌شوند تا بتوان روی آن‌ها محاسبات آماری انجام داد.

# بخش ۴: مرتب‌سازی داده‌های ستون

### هدف این بخش

آماده‌سازی داده‌ها برای محاسبه میانه و چارک‌ها.
# بخش ۵: محاسبه میانه (Median)

### هدف این بخش

به‌دست آوردن مرکز توزیع هر ستون.
### توضیح

بعد از مرتب‌سازی، عنصر وسط آرایه به‌عنوان میانه ستون در نظر گرفته می‌شود.
# بخش ۶: محاسبه IQR فاصله بین چارک‌ها


محاسبه پراکندگی داده‌ها به‌صورت مقاوم در برابر outlier و داده های پرت.
# بخش ۷: اعمال Robust Scaling روی داده‌ها

### هدف این بخش

نرمال‌سازی داده‌ها برای استفاده در رگرسیون.



اگر IQR صفر باشد (ستون ثابت):

- مقدار ستون صفر می‌شود تا تقسیم بر صفر رخ ندهد.
